## 技能与装备目录补充核对

- 线上技能选择器先按稀有度降序，同稀有度按原始 `skills.json` 数据逆序；离线目录现由 Rust 保留并透传 `sourceOrder`，不再用中文名称或 `rank` 猜测排序。
- 线上装备先按阶数降序，同阶按装备解锁等级降序，最后保留原始 `items.json` 顺序；离线装备选择器已采用相同规则，并按英雄等级隐藏超过“最大装备阶数”的装备。
- 装备的 `elementAffinity` / `spiritAffinity` 与内置 `lTag2` / `lTag3` 均由 Rust 规则引擎按线上规则计算；内置附魔自动生效并按 1.5 倍计算，离线六槽卡片也显示“元素亲和/精萃亲和”标记。
